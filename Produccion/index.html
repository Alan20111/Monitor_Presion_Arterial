Acerca del proyecto

Aplicación de escritorio en JavaFX (tema BootstrapFX) con base de datos MySQL para el monitoreo clínico de la presión arterial. Implementa el patrón MVC con capa DAO y una conexión Singleton a la base de datos.

Características

  • CRUD de pacientes y de mediciones.
  • Diagnóstico automático de IMC (bajo peso, normal, sobrepeso, obesidad).
  • Clasificación de presión arterial (Normal, Elevada, Hipertensión 1/2, Crisis).
  • Filtro dinámico: al elegir un paciente se muestran solo sus mediciones.
  • Validaciones y alertas de confirmación.
